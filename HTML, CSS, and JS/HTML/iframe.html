Iframes

An iframe is like a little window that has been placed into your webpage. The term iframe is an abbreviation for "inline frame".

One common use of iframes is to embed Google Maps into the webpage. The content of the iframe can be any HTML page (either located on the web servers or external sites).




Another common use for iframes is to embed videos into a page. Video content is often included to support other content on page.

This works fine if you have an externally hosted video. However, if I have my own video content that is located on the same web server as this document, then the HTML5 <video> tag to accomplish this.
